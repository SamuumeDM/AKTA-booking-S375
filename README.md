# AKTA 使用预约登记（网页版）

这是一个纯前端网页版本，不需要安装 Node.js，也不需要打包 App。

## 用法

### 本地直接打开
1. 解压这个文件夹
2. 双击 `index.html`
3. 浏览器里就能使用

### 功能
- 单台 AKTA 预约登记
- 时间冲突检测
- 编辑 / 删除预约
- CSV 导出
- 本地离线保存（浏览器 localStorage）
- 自定义标题、默认整理时间、提醒文字

## 注意
- 数据保存在当前浏览器本地。
- 换浏览器、清缓存、换电脑后，原数据不会自动同步。
- 如果后面想多人共用，需要再接数据库或表单后端。

## 部署成真正网页
这个项目是静态网站，可以直接部署到静态托管平台。
- GitHub Pages：适合放到 GitHub 仓库后发布
- Netlify：支持手动拖拽整个文件夹部署
